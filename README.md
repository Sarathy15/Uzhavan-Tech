## 🌾 AI-Powered Agricultural Platform

Uzhavan-Tech is a comprehensive AI-driven platform designed to empower farmers and agricultural professionals with cutting-edge technology for crop management, disease detection, weather monitoring, and intelligent assistance.

## ✨ Features

- **Disease Detection**: Upload crop images to detect diseases using advanced AI
- **Weather Dashboard**: Real-time weather information and forecasts for better planning
- **Analytics Dashboard**: Insights and analytics for crop performance
- **AI Chatbot**: Intelligent assistant for agricultural queries and advice
- **Multi-language Support**: Interface available in multiple languages
- **Image Upload & Management**: Secure image storage and processing with ImageKit
- **Authentication**: Secure user authentication with Firebase

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Node.js, Express
- **AI/ML**: Google Gemini API
- **Database**: Firebase
- **Image Processing**: ImageKit
- **Charts**: Recharts
- **Styling**: CSS

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sarathy15/Uzhavan-Tech.git
   cd Uzhavan-Tech
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add the following:
   ```
   GEMINI_API_KEY=your_gemini_api_key_here
   # Add other required API keys (Firebase, ImageKit, etc.)
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. (Optional) Start the ImageKit signer server:
   ```bash
   npm run start-signer
   ```

## 📱 Usage

1. Open your browser and navigate to `http://localhost:5173`
2. Sign up or log in using Firebase authentication
3. Explore the various features:
   - Upload crop images for disease detection
   - Check weather forecasts
   - Chat with the AI assistant
   - View analytics

## 🏗️ Build & Deploy

To build the project for production:

```bash
npm run build
npm run preview
```
