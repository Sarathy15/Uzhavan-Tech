# 🌾 Uzhavan-Tech

### AI-Powered Agricultural Platform

Uzhavan-Tech is an AI-driven web application that helps farmers and agricultural professionals make smarter decisions through crop disease detection, weather insights, analytics, and an intelligent chatbot.

---

## 🚀 Live Demo

👉 https://uzhavan-tech.netlify.app/


## 📌 Overview

This platform combines **AI, real-time data, and interactive dashboards** to simplify agricultural decision-making and improve productivity.



## ✨ Features

* 🌱 AI-based crop disease detection from images
* 🌦️ Real-time weather updates and forecasts
* 📊 Interactive analytics dashboard
* 🤖 AI chatbot for agricultural assistance
* 🌍 Multi-language support
* 🔐 Secure authentication (Firebase)
* 🖼️ Image upload & management (ImageKit)


## 🛠️ Tech Stack

* **Frontend:** React, TypeScript, Vite
* **Backend:** Node.js, Express
* **AI:** Google Gemini API
* **Database:** Firebase
* **Image Handling:** ImageKit
* **Charts:** Recharts

## ⚙️ Setup

```bash
git clone https://github.com/Sarathy15/Uzhavan-Tech.git
cd Uzhavan-Tech

```
